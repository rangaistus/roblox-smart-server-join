Privacy Policy for Roblox Smart Server Join

Last Updated: July 26, 2025
Extension Name: Roblox Smart Server Join
Version: 1.0

1. Introduction

This Privacy Policy describes how the "Roblox Smart Server Join" Chrome extension ("we", "our", or "the extension") collects, uses, and protects your information when you use our browser extension. We are committed to protecting your privacy and ensuring transparency about our data practices.

2. Information We Collect

2.1 User Preferences

  • Language Preference: We store your selected language preference (English, Turkish, or Dutch) locally in Chrome's sync storage to provide a personalized experience across your devices.

2.2 Roblox Data Access

When you use the smart server join feature, the extension accesses the following information from Roblox:

  • User Authentication Status: We check if you're logged into Roblox to enable functionality
  • User ID: Retrieved from Roblox's authenticated user endpoint to identify your account
  • Friends List: We access your Roblox friends list to find servers without your friends
  • Server Information: We retrieve public server data (player count, ping, server IDs) to find optimal servers

2.3 Technical Data

  • Game Page URLs: We process Roblox game page URLs to extract game IDs
  • Server Performance Data: We analyze server ping and player counts for optimization

3. How We Use Your Information

3.1 Primary Functions

  • Find and join Roblox servers with low ping and without your friends
  • Provide multilingual interface based on your language preference
  • Optimize server selection based on performance metrics

3.2 Technical Operations

  • Interface with Roblox's official APIs to gather server information
  • Inject necessary scripts to facilitate server joining through Roblox's launcher
  • Store language preferences for consistent user experience

4. Data Storage and Security

4.1 Local Storage

  • Language preferences are stored locally using Chrome's sync storage
  • No personal Roblox data is permanently stored by our extension
  • All Roblox data is processed in real-time and not retained

4.2 Data Transmission

  • All communications with Roblox services use HTTPS encryption
  • We only access Roblox data through official, public APIs
  • No data is transmitted to third-party servers outside of Roblox

5. Third-Party Services

5.1 Roblox Integration

This extension interacts with Roblox services including:

  • users.roblox.com: For user authentication verification
  • friends.roblox.com: For accessing friends list data
  • games.roblox.com: For retrieving server information
  • www.roblox.com: For game page integration and launcher interaction

These interactions are governed by Roblox's Privacy Policy.

6. Permissions Explained

6.1 Required Permissions

  • activeTab: Allows the extension to interact with the currently active Roblox game page
  • scripting: Enables injection of scripts necessary for server joining functionality
  • storage: Permits saving your language preference locally
  • Host permissions for Roblox domains: Allows communication with Roblox APIs for core functionality

7. Data Sharing and Disclosure

We do not sell, trade, or share your personal information with third parties.
  • We do not collect personal information beyond what's necessary for functionality
  • We do not share data with advertisers or marketing companies
  • We do not track user behavior across websites
  • All data processing occurs locally within your browser

8. Children's Privacy

Our extension is designed to work with Roblox, which is used by people of all ages including children under 13. We do not knowingly collect personal information from children. The extension only processes publicly available server data and user preferences. Parents and guardians should review this policy and supervise their children's use of browser extensions.

9. Your Rights and Choices

9.1 Data Control

  • Language Preference: You can change or reset your language preference at any time through the extension popup
  • Extension Removal: You can uninstall the extension at any time, which will remove all stored preferences
  • Storage Access: You can clear extension data through Chrome's extension management interface

9.2 Roblox Data

Since we access Roblox data through their APIs, your rights regarding that data are governed by Roblox's privacy policy. You maintain full control over your Roblox account and can manage privacy settings directly on the Roblox platform.

10. Updates to This Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in our practices or for legal, operational, or regulatory reasons. When we make changes:

  • We will update the "Last Updated" date at the top of this policy
  • Significant changes will be communicated through extension updates
  • Continued use of the extension after updates constitutes acceptance of the revised policy
